Sobre a Disciplina

O que é a disciplina

Desenvolvimento Web é a matéria que ensina como criar páginas pra internet. A gente usa o HTML pra montar a estrutura da página — textos, imagens, listas, formulários — e o CSS pra deixar tudo com uma aparência melhor, mexendo em cores, fontes e espaçamentos.

Por que ela é importante

HTML e CSS são a base de praticamente qualquer site. Mesmo usando frameworks mais modernos depois, você precisa entender isso primeiro. Fora que é uma das coisas mais pedidas quando a gente vai procurar estágio ou emprego na área.

O que a gente viu na matéria
  • Estrutura básica de uma página HTML
  • Estilização com CSS
  • Uso do Bootstrap pra deixar o site responsivo
